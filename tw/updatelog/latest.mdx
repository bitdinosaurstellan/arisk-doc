# 版本 2.0.0
**发布日期: 2025年12月30日**

<Note>
Arisk 正式进入 V2 时代！本次更新带来了全新的 API 架构演进，我们在接口效能、管理便捷性及分析深度上实现了质的飞跃。
</Note>

## 🚀 Arisk V2 API 全新发布

V2 版本不仅是简单的版本号迭代，更是我们对区块链风控与地址监控能力的重构。

### 1. 深度 RESTful 架构演进
V2 API 采用资源导向设计，全面引入路径参数（Path Parameters），使接口语义更加清晰直观。相比 V1 的扁平化路径，V2 的层次化结构让开发者能更自然地操作特定资源，同时也提升了 API 的可扩展性与标准化程度。

### 2. 批量操作效能质变
针对企业级大规模地址管理场景，V2 引入了全新的批量操作接口：
- **批量添加**：支持单次请求添加多个监控地址，大幅减少 API 调用频率。
- **批量管理**：通过统一的删除接口高效清理监控列表，简化维护工作量。
这些改进显著降低了大规模集成时的网络往返次数与系统处理时耗。

### 3. 全新异步分析任务系统
我们将原有的查询功能全面升级为“分析任务（Analysis Task）”引擎：
- **更强的异步处理能力**：支持更复杂、更高并发的分析请求，任务状态查询更加稳定可靠。
- **更深度的评估维度**：V2 提供了比 V1 更详尽的风险权重分解与链上行为画像，帮助您更精准地识别潜在风险。
- **任务链路追踪**：每个分析任务都有唯一的任务 ID，方便您进行历史回溯、审计与结果持久化。

### 4. 精细化监控配置
V2 优化了地址监控的状态管理逻辑，支持更灵活的监控开关切换（如：启用/暂停）。同时，我们优化了警报过滤机制，确保警报通知更加精准。

### 5. 国际化体验升级
- **全方位文档覆盖**：完成了 V1 与 V2 API 的中英文档全面覆盖，填补了英文文档空白。
- **配置标准化**：OpenAPI 定义文件中现已包含官方生产地址 `https://api.arisk.io`，支持更便捷的在线调试。
