---
title: "Get User Profile (v2)"
api: "GET /v2/user/profile"
description: "Get detailed user profile including usage statistics and quotas"
---

## Request Headers

<ParamField header="Authorization" type="string" required>
  Bearer token authentication
</ParamField>

## Response Example

```json
{
  "user_id": "usr_123456",
  "email": "user@example.com",
  "created_at": "2025-01-01T00:00:00Z",
  "subscription": {
    "plan": "professional",
    "status": "active",
    "expires_at": "2026-01-01T00:00:00Z"
  },
  "usage": {
    "api_calls_this_month": 1250,
    "api_calls_limit": 10000,
    "addresses_monitored": 45,
    "addresses_limit": 100
  },
  "preferences": {
    "language": "en",
    "timezone": "UTC",
    "notifications_enabled": true
  }
}
```

<Info>
  v2 API includes subscription info, usage statistics, and user preferences
</Info>
