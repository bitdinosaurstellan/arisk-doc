---
title: "Analyze Address Risk (v1)"
api: "POST /v1/analyze/address"
description: "Analyze risk level and details for a single blockchain address"
---

## Request Parameters

<ParamField body="address" type="string" required>
  The blockchain address to analyze
</ParamField>

<ParamField body="chain" type="string" required>
  Blockchain network (e.g., ethereum, bitcoin, polygon)
</ParamField>

## Response Example

```json
{
  "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
  "risk_score": 75,
  "risk_level": "high",
  "labels": ["mixer", "suspicious"]
}
```

<Note>
  This is v1 API. We recommend using v2 API for more detailed risk analysis
</Note>
