## Get API Key

To start using Arisk, you need an API key.

<Steps>
  <Step title="Create Account">
    Visit [api-key](https://arisk.io/console/api-key) and register for a free account.
  </Step>
  
  <Step title="Generate API Key">
    Navigate to **Settings > API Keys** and click "Create New Key". Please securely save this key - you will not be able to view it again.
  </Step>
  
  <Step title="Set Authentication">
    Include your API key in all requests using the `X-API-Key` header:
    
    ```bash
    curl https://api.arisk.com/v1/analyze \
      -H "X-API-Key: YOUR_API_KEY"
    ```
  </Step>
</Steps>

## First Risk Analysis

Let's analyze a wallet address to evaluate its risk level.

### Analyze Wallet Address

<CodeGroup>

```bash cURL
curl -X POST https://api.arisk.com/v1/analyze/address \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "chain": "ethereum"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.arisk.com/v1/analyze/address', {
  method: 'POST',
  headers: {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    address: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
    chain: 'ethereum'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://api.arisk.com/v1/analyze/address"
headers = {
    "X-API-Key": "YOUR_API_KEY",
    "Content-Type": "application/json"
}
payload = {
    "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "chain": "ethereum"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "net/http"
    "fmt"
)

func main() {
    url := "https://api.arisk.com/v1/analyze/address"
    
    payload := map[string]string{
        "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
        "chain": "ethereum",
    }
    
    jsonData, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonData))
    req.Header.Set("X-API-Key", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")
    
    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()
}
```

## Understanding Risk Scores

Arisk assigns a **0 to 100** risk score to each address:

| Risk Score | Risk Level | Description |
|----------|----------|-------------|
| 0-20 | üü¢ **Low** | Legal activities, low risk |
