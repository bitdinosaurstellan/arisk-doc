---
title: "分析地址风险 (v1)"
api: "POST /v1/analyze/address"
description: "分析单个区块链地址的风险等级和详细信息"
---

## 请求参数

<ParamField body="address" type="string" required>
  要分析的区块链地址
</ParamField>

<ParamField body="chain" type="string" required>
  区块链网络（例如：ethereum, bitcoin, polygon）
</ParamField>

## 响应示例

```json
{
  "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
  "risk_score": 75,
  "risk_level": "high",
  "labels": ["mixer", "suspicious"]
}
```

<Note>
  这是 v1 API，推荐使用 v2 API 以获得更详细的风险分析
</Note>
