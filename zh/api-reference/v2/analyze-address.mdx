---
title: "分析地址风险 (v2)"
api: "POST /v2/analyze/address"
description: "分析单个区块链地址的风险等级和详细信息，包含更全面的风险维度"
---

## 请求参数

<ParamField body="address" type="string" required>
  要分析的区块链地址
</ParamField>

<ParamField body="chain" type="string" required>
  区块链网络（例如：ethereum, bitcoin, polygon, bsc）
</ParamField>

<ParamField body="include_transactions" type="boolean" default="false">
  是否包含最近的交易记录
</ParamField>

<ParamField body="depth" type="integer" default="1">
  分析深度（1-5），数值越大分析越深入
</ParamField>

## 响应示例

```json
{
  "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
  "risk_assessment": {
    "overall_score": 75,
    "risk_level": "high",
    "confidence": 0.92
  },
  "risk_categories": {
    "mixer": {
      "score": 85,
      "evidence": ["Connected to known mixer services"]
    },
    "fraud": {
      "score": 65,
      "evidence": ["Involved in suspicious transactions"]
    }
  },
  "labels": ["mixer", "high-risk", "suspicious-activity"],
  "last_updated": "2025-12-26T08:50:00Z"
}
```

<Tip>
  v2 API 提供了更细粒度的风险分类和证据链，建议在生产环境使用
</Tip>
