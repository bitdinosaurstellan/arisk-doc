---
title: "获取用户资料 (v2)"
api: "GET /v2/user/profile"
description: "获取当前用户的详细资料信息，包括使用统计和配额"
---

## 请求头

<ParamField header="Authorization" type="string" required>
  Bearer token 认证
</ParamField>

## 响应示例

```json
{
  "user_id": "usr_123456",
  "email": "user@example.com",
  "created_at": "2025-01-01T00:00:00Z",
  "subscription": {
    "plan": "professional",
    "status": "active",
    "expires_at": "2026-01-01T00:00:00Z"
  },
  "usage": {
    "api_calls_this_month": 1250,
    "api_calls_limit": 10000,
    "addresses_monitored": 45,
    "addresses_limit": 100
  },
  "preferences": {
    "language": "zh",
    "timezone": "Asia/Shanghai",
    "notifications_enabled": true
  }
}
```

<Info>
  v2 API 新增了订阅信息、使用统计和用户偏好设置
</Info>
